# Code of Conduct

## Our Pledge
We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards
- Be respectful and constructive
- Assume good intent
- Focus feedback on code and behavior, not people
- Accept responsibility and learn from mistakes

## Unacceptable Behavior
- Harassment, threats, or discrimination
- Personal attacks or trolling
- Publishing private information without permission

## Enforcement
Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

## Reporting
Report incidents privately via the contact channel listed in [SECURITY.md](SECURITY.md).
